# LLM 기반 사내 지식관리 시스템 (TechTree Project)

**LLM(대규모 언어 모델)을 활용하여 기업의 사내 지식 관리 및 업무 자동화**를 구현한 프로젝트. 2025 KNU 직무마스터 프로젝트의 일환으로 (주)루트랩과 협업하여 진행함.

---

## 프로젝트 개요
기업 내부의 방대한 문서와 지식을 효율적으로 관리하고, 보고서 작성 등의 반복적인 업무를 자동화하여 **인건비 절감 및 생산성 향상**이 목표.

##  주요 핵심 기능
* **LLM 기반 지식 검색:** 사내 데이터를 벡터화하여 정확도 높은 답변을 제공하는 지식관리 시스템.
* **보고서 자동화:** AI를 활용하여 외주 제안서 및 업무 보고서 작성 시간 30~40% 단축.
* **ROI 최적화:** 자동화 도입을 통한 인건비 20~24% 절감 효과 및 초기 투자 비용의 빠른 회수.
* **프롬프트 엔지니어링:** (주)루트랩 맞춤형 OpenAI API 프롬프트 설계로 비즈니스 특화 응답 구현.

##  기술 스택
* **Core:** LLM (OpenAI API), Vector Search
* **Development:** Python, OpenAI API, Prompt Engineering
* **Business Logic:** SI(System Integration) 기반 비즈니스 프로세스 자동화


##  기대 효과 및 성과
* **업무 생산성 향상:** 반복적인 문서 작업 시간 평균 2~4시간에서 대폭 단축.
* **경제적 가치:** 직원 1인당 연간 약 700~1,000만 원 이상의 인건비 절감 효과 실현.
* **실무 최적화:** 기업의 실제 문제를 해결하는 프로세스로 설계되어 즉각적인 도입 가능.
